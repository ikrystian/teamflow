### Zadania:

  - Obsługuje projects, session, hideProjectSelect

### Dashboard:
- src/components/dashboard/content.tsx
  - Używa TasksTable do wyświetlania wszystkich zadań z systemu
  - Pobiera dane z API: tasks, users, taskStatuses
  - Obsługuje aktualizację zadań przez inline editing

- src/components/dashboard/tasks-table.tsx
  - Główny komponent tabelki zadań w stylu Asana
  - Używa TanStack Table do sortowania, filtrowania, paginacji

  - Obsługuje inline editing przez EditableCell
  - Kolumny: nazwa, osoba przypisana, priorytet, data, status, projekt (z kolorami), data utworzenia, szacowany czas
  - Nazwy projektów wyświetlane w kolorach projektów
  - Kolumna daty utworzenia domyślnie ukryta
  - Wiersze nagłówków grup z tłem bg-muted/30

- src/components/dashboard/editable-cell.tsx
  - Komponent do edycji inline różnych typów danych
  - Obsługuje: text, date, user, priority, status, select
  - Automatyczne zapisywanie po stracie focusa lub Enter
  - Placeholdery dla pustych wartości
  - Statusy wyświetlane bez kolorów (tylko tekst)
  - Użytkownicy wyświetlani jako avatar z tooltip (imię po najechaniu)

## SYSTEM BREADCRUMBS
==========================================

### Komponent DashboardBreadcrumbs
- **Lokalizacja**: src/components/dashboard/breadcrumbs.tsx
- **Integracja**: Dodany do DashboardLayout pod headerem
- **Funkcjonalność**: Automatyczne generowanie ścieżki nawigacji na podstawie URL

### Relacje z innymi komponentami:
- **DashboardLayout**: Główny layout zawierający breadcrumbs
- **shadcn/ui Breadcrumb**: Wykorzystuje komponenty Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink, BreadcrumbPage, BreadcrumbSeparator
- **usePathname**: Hook Next.js do pobierania aktualnej ścieżki URL
- **API endpoints**: Pobiera dane z /api/projects, /api/teams, /api/users dla dynamicznych nazw

### Mapowanie segmentów URL:
- tasks → "Moje zadania"
- teams → "Zespoły"
- reports → "Raporty"
- calendar → "Kalendarz"
- projects → "Projekty"
- profile → "Profil"
- settings → "Ustawienia"
- info → "Informacje"
- UUID-like segments → Rzeczywiste nazwy z API (projekty, zespoły, użytkownicy)

### Funkcje breadcrumbs:
1. Automatyczne wykrywanie ścieżki na podstawie pathname
2. Tłumaczenie segmentów na polskie nazwy
3. Pobieranie rzeczywistych nazw dla dynamicznych segmentów
4. Zawsze widoczne breadcrumbs (również na głównej stronie dashboardu)
5. Responsywny design z ikoną Home dla pierwszego elementu
