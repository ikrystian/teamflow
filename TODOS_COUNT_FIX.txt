NAPRAWA NIEPRAWIDŁOWEJ LICZBY ZADAŃ Z TODOLIST W SZCZEGÓŁACH PROJEKTU
=====================================================================

PROBLEM:
--------
W szczegółach projektu wyświetlana była nieprawidłowa liczba zadań z todolist.
Liczba zadań nie uwzględniała todos, co powodowało rozbieżności w statystykach.

PRZYCZYNA:
----------
1. W API endpoint `/api/projects/[projectId]/route.ts` nie były dołączane todos przy pobieraniu zadań
2. W komponencie `project-details-content.tsx` statystyki były obliczane ale nie przypisywane do zmiennej
3. Brak wyświetlania statystyk zadań w widoku szczegółów projektu

ROZWIĄZANIE:
------------

1. **Naprawiono API endpoint** (`src/app/api/projects/[projectId]/route.ts`):
   - Dodano `todos: true` do include w zapytaniu o zadania
   - Teraz todos są prawidłowo dołączane do danych zadań

2. **Naprawiono komponent szczegółów projektu** (`src/components/projects/project-details-content.tsx`):
   - Poprawiono przypisanie wyniku `getTaskStats()` do zmiennej `stats`
   - Dodano wyświetlanie statystyk zadań w formie kart na górze strony
   - Statystyki pokazują: wszystkie zadania, ukończone, w toku, zaległe

3. **Dodano test weryfikacyjny** (`test-todos-count.js`):
   - Test sprawdza czy todos są prawidłowo dołączane
   - Porównuje wyniki przed i po naprawie
   - Potwierdza poprawność działania

WYNIKI:
-------
- Todos są teraz prawidłowo dołączane do danych zadań
- Statystyki zadań są poprawnie obliczane i wyświetlane
- Dodano wizualne karty ze statystykami w widoku szczegółów projektu
- Test potwierdza poprawność naprawy

PLIKI ZMIENIONE:
---------------
1. src/app/api/projects/[projectId]/route.ts - dodano todos: true
2. src/components/projects/project-details-content.tsx - naprawiono stats i dodano wyświetlanie
3. test-todos-count.js - test weryfikacyjny (nowy plik)
4. TODOS_COUNT_FIX.txt - dokumentacja naprawy (ten plik)

RELACJE:
--------
- Todos są powiązane z zadaniami (Task) przez taskId
- Każde zadanie może mieć wiele todos
- Todos mają status isCompleted do śledzenia postępu
- Statystyki uwzględniają todos w obliczeniach postępu zadań

TESTOWANIE:
-----------
1. Uruchom test: `node test-todos-count.js`
2. Sprawdź aplikację w przeglądarce na http://localhost:3001
3. Przejdź do szczegółów projektu z todos
4. Sprawdź czy statystyki są poprawnie wyświetlane
