# Mono Theme Implementation - TeamFlow

## Przegląd
Implementacja monochromatycznego motywu (mono theme) w aplikacji TeamFlow z wykorzystaniem najnowszej wersji shadcn/ui.
Motyw wykorzystuje odcienie szarości dla spójnego, minimalistycznego wyglądu.

## Zmiany w kolorach CSS

### Light Mode (Jasny motyw)
- **Background**: oklch(0.99 0 0) - bardzo jasny szary
- **Foreground**: oklch(0.09 0 0) - bardzo ciemny szary
- **Primary**: oklch(0.09 0 0) - ciemny szary dla głównych elementów
- **Secondary**: oklch(0.96 0 0) - jasny szary dla drugorzędnych elementów
- **Muted**: oklch(0.96 0 0) - wyciszony szary
- **Border**: oklch(0.89 0 0) - średni szary dla obramowań
- **Radius**: 0.5rem - zmniejszony promień zaokrąglenia dla ostrzejszego wyglądu

### Dark Mode (Ciemny motyw)
- **Background**: oklch(0.09 0 0) - bardzo ciemny szary
- **Foreground**: oklch(0.95 0 0) - bardzo jasny szary
- **Card**: oklch(0.12 0 0) - ciemny szary dla kart
- **Primary**: oklch(0.95 0 0) - jasny szary dla głównych elementów
- **Secondary**: oklch(0.18 0 0) - średnio ciemny szary
- **Border**: oklch(0.18 0 0) - ciemny szary dla obramowań

## Ulepszenia okienka dodawania zadania

### Zmiany w układzie
1. **Zwiększona szerokość**: max-w-[700px] zamiast max-w-[625px]
2. **Lepsze przewijanie**: max-h-[90vh] overflow-y-auto
3. **Ulepszone odstępy**: space-y-6 dla głównego kontenera, space-y-4 dla pól

### Ulepszenia wizualne
1. **Lepsze etykiety**: Dodano gwiazdki (*) dla wymaganych pól
2. **Większe pola**: height: h-10 dla wszystkich pól wyboru
3. **Ulepszone selecty**: 
   - Dodano ikony dla priorytetów (kolorowe kropki)
   - Lepsze wyświetlanie członków zespołu z awatarami
   - Ulepszone wyświetlanie projektów z nazwami zespołów

### Sekcja załączników
1. **Lepsze tło**: bg-muted/30 z border-dashed
2. **Efekty hover**: scale-105 dla obrazków
3. **Lepsze pozycjonowanie**: przycisk usuwania poza obrazkiem
4. **Cienie**: shadow-sm dla obrazków, shadow-md dla przycisków

### Przyciski i błędy
1. **Lepsze błędy**: tło z obramowaniem w kolorze destructive
2. **Spinner ładowania**: animowany spinner zamiast tekstu
3. **Responsywne przyciski**: flex-col-reverse sm:flex-row

## Ulepszenia Theme Toggle
- Dodano ikony dla każdej opcji motywu
- Lepsze wizualne rozróżnienie opcji
